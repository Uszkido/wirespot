# WireSpot Privacy Policy

Effective date: 2026-07-14

WireSpot is a MikroTik RouterOS hotspot management application by Vexel
Innovations. This policy explains what information the app stores, why it is
needed, and how operators control sharing.

## Summary

WireSpot is designed as an operator tool. Most operational data is stored on
the Android device and is used to manage routers, vouchers, printers, reports,
VPN settings, and app access.

WireSpot does not sell personal data.

## Data Stored On The Device

WireSpot may store:

- router profiles
- router connection settings
- hotspot voucher history
- sales records
- printer profiles
- app settings
- language/currency preferences
- professional co-branding details
- WireGuard tunnel settings
- trial/license status
- backup preview data generated by the operator

Router credentials and sensitive local secrets are stored through encrypted
device storage where supported by Android.

## Router Communication

When an operator connects WireSpot to a MikroTik router, the app sends RouterOS
API commands required for the selected action. Examples include:

- reading router status
- loading hotspot users and sessions
- creating or updating hotspot users
- creating vouchers
- disconnecting sessions
- reading profiles, queues, cookies, and IP bindings
- applying hotspot setup actions selected by the operator

WireSpot does not perform hidden router management actions outside the operator
workflow.

## VPN Data

WireSpot can import WireGuard tunnel configuration so the operator can connect
to private router networks. The configuration is used to establish the VPN
tunnel requested by the operator.

## Bluetooth And Printing

WireSpot requests Bluetooth access to list paired thermal printers and print
voucher receipts.

## Camera

WireSpot requests camera access only for scanning WireGuard QR configurations.

## Network Access

WireSpot uses network access to communicate with routers and VPN networks. In
future production builds, network access may also be used for license
validation or billing verification.

## Reports, Receipts, And Backups

WireSpot can generate voucher receipts, report exports, and backup JSON. These
are shared only when the operator explicitly uses app share/export controls or
copies backup content.

Operators are responsible for protecting exported files and backup JSON.

## Data Sharing

WireSpot does not sell personal data. WireSpot does not share reports,
receipts, backups, or router data unless the operator chooses to export or
share them.

## Data Removal

Operators can remove data by deleting routers, vouchers, printers, and app data
from the app or Android system settings. Removing the app may remove local app
data depending on Android backup settings.

## Security Practices

Operators should:

- use dedicated RouterOS accounts for WireSpot
- avoid exposing RouterOS API ports publicly
- use WireGuard for remote management
- protect device PIN/password
- rotate router credentials if a phone is lost
- protect exported backup JSON
