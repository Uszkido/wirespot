# WireSpot Privacy Policy

Effective date: 2026-07-13

WireSpot is a MikroTik RouterOS hotspot management application by Vexel Innovations.

## Data Stored On The Device

WireSpot stores router profiles, hotspot voucher history, sales records, printer settings, app settings, WireGuard tunnel settings, and license status locally on the device.

Router credentials and sensitive local secrets are stored using encrypted device storage where supported by Android.

## Router Communication

When you connect WireSpot to a MikroTik router, the app sends RouterOS API commands needed for the actions you request, such as reading router status, managing hotspot users, creating vouchers, disconnecting sessions, and configuring hotspot settings.

## VPN Data

WireSpot can import and use WireGuard tunnel configuration to connect to private router networks. VPN configuration is used only to establish the tunnel requested by the operator.

## Bluetooth And Printing

WireSpot requests Bluetooth access to discover paired thermal printers and print voucher receipts.

## Camera

WireSpot requests camera access only for scanning WireGuard QR configurations.

## Internet And Network Access

WireSpot uses network access to communicate with routers and, in future production builds, may use network access for license validation or billing verification.

## Data Sharing

WireSpot does not sell personal data. Report exports, voucher receipts, and backup previews are shared only when the operator explicitly uses the app's share/export controls.
